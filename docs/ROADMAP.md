# Roadmap

> Checklist-based tracking for Sheety CRM development.

---

## Phase 1: Core Parity
*Target: 4 weeks*

### Search & Navigation
- [ ] Command palette (⌘K) with instant search
- [ ] Keyboard shortcuts (N=new, E=edit, D=delete)
- [ ] Quick-add lead modal

### Activity Tracking
- [ ] Notes system (Markdown support)
- [ ] Activity timeline on lead/contact pages
- [ ] Tasks with due dates and reminders
- [ ] Task notifications

### Filtering & Views
- [ ] Multi-condition filter builder
- [ ] Date range filters
- [ ] Saved views (Smart Views)
- [ ] View sharing

## Phase 2: Power Features
*Target: 4 weeks*

### Customization
- [ ] Custom fields (text, number, date, select, multi-select)
- [ ] Custom field UI configuration
- [ ] Field validation rules

### Bulk Operations
- [ ] Multi-select rows
- [ ] Bulk status update
- [ ] Bulk delete
- [ ] Bulk assign owner

### Data Management
- [ ] Lead merge with conflict resolution
- [ ] Duplicate detection
- [ ] Enhanced dashboard with charts
- [ ] Pipeline value summary
